# System Design Interview - Go Implementations

Go implementations of system design patterns from "System Design Interview" by Alex Xu.

## Projects

- **Consistent Hashing**: Distributed load balancing with virtual nodes
- **Rate Limiter**: HTTP middleware with fixed window and sliding window algorithms

## Quick Start

```bash
go run consistenthashing/main.go
go run ratelimiter/cmd/*.go
```

Requires Go 1.26+
